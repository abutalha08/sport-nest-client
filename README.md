# 🏟️ SportNest – Sports Facility Booking Management System

**SportNest** is a full-stack MERN-based sports facility booking platform where users can explore, book, and manage sports venues such as football turfs, badminton courts, swimming lanes, and tennis courts. Facility owners can also create and manage their own listings.

The platform is designed to simulate a real-world sports reservation system with authentication, role-based access, and dynamic booking workflows.

## 🎯 Project Purpose

SportNest is built to demonstrate a real-world booking system where:

- Users can browse and book sports facilities
- Facility owners can manage their listings
- All bookings are securely stored in the database
- Authentication and authorization are properly implemented

This project showcases a complete **MERN stack application with CRUD operations, authentication, and role-based access control.**

---

## ✨ Key Features

### 👤 User Features
- Secure user registration and login system  
- Google authentication (via Google Cloud OAuth)  
- Browse all available sports facilities  
- View detailed facility information  
- Book facilities by selecting date and time  
- View and cancel bookings  

---

### 🏟️ Facility Management
- Add new sports facilities (protected route)  
- Update facility details  
- Delete facility (restricted to owner)  
- Dedicated dashboard for managing listings  

---

### 📦 Booking System
- Real-time facility booking with time slots  
- Automatic price calculation based on duration  
- Booking status tracking (pending / cancelled)  
- User-specific booking history  

---

### 🔐 Authentication & Security
- JWT-based authentication system  
- HTTP-only cookie sessions for security  
- Protected private routes  
- Role-based access control (User / Owner)  

---

### 🎨 UI/UX Features
- Fully responsive design (mobile, tablet, desktop)  
- Modern UI built with Tailwind CSS  
- Toast notifications using React Hot Toast  
- Smooth loading states and transitions  
- Custom 404 error page
